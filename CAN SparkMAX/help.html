CAN SparkMAX

What is it?

The SparkMAX speed controller uses CAN protocol to control the the attached motor. It supports both Brushed and the Brushless NEO Motor. Set the mode on the SparkMAX using the REV SPARK MAX Client or using the MODE button. See the Quickstart Guide for detailed info.

Properties

CAN ID
The ID of the CAN device. This must match the ID of the device configured with the SPARK MAX Client.
Motor Type
Select Brushed or Brushless Motor type. Only Brushless motors will use the white wire.
